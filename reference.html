Reference

Tandem has a library that offers over 30 operations for building a wide range of interactive Web enhancements.

Pick

Select / find / load information from the Web.
  • Extract Element
    • Extract elements at [selector] from [source].
    Extract DOM elements from the [source] node's values. As [selector] would be a very long query string, it is almost impossible to specify the query directly. Instead, set up the current node values to be the elements within the source. Tandem will suggest a correct [selector].
    Show Demo video
  • Extract Parent
    • Get enclosing elements of [source], [step]-step above.
    Select parent elements of the [source] node. For example, when the source node contains the "Gas"(SPAN tag) in the figure below, Extract Parent with 1,2,and 3-steps will yield higher parent elements, TD, TR, and TBODY respectively.
  • Find Tab
    • Find a currently open [url], and execute the following nodes in the tab.
    Not available in this version.
  • Get Attribute
    • Get [key] of [source].
    [source] node can hold any page elements.
    [key] must be a property name of the element in the [source] node.
    The following table lists up the keys supported in Tandem.
    key description applicable tags
    text text content within a page element. All
    value value within <select> or <input> tags. <SELECT>, <INPUT>
    checked Either checkbox is checked or not. <INPUT type='checkbox'>
    download File name of downloadable links. <A>
    visibility Setting it to 'none' will make the element invisible. All
    color Text color can be any standard color names (e.g. red, green) listed here, or valid RGB values. All
    background-color Background color of element allows the same rule as color. All
    source 'src' attribute of image tag. <IMG>
    link URL of clickable link. <A>
  • Load Page
    • Load pages of URLs in [source] using [mode].
    Too complex to explain now.
  • Substring
    • Get part of text in [source] from [from] to [to], including from([include_from]]) and to([include_to]).
    ...

Transform

Transforms a list of values to another form.
  • Arithmetic
    Calculate [operand_A] [operator] [operand_B]
  • Compose Text
    Compose each pair of [text_A] and [text_B] connected by [connector].
  • Count
    Count [source].
  • Filter
    Filter items in [items] by [true_or_false] of [booleans].
  • Join
    Join all the inputs.
  • Literal
    Directly set the current node data to [source].
  • Literal Element
    Create a new element of [jsonML], which was extracted from exiting Web elements.
  • Number Test
    Evaluate [operand_A] [operator] [operand_B].
  • Sort
    Sort [source] in [direction]-order.
  • String Test
    Distinguish whether [key] is [isIn] [source] .
  • Sum
    Add all numbers in [source].
  • Unique
    Get list of unique elements of [source].

Apply

Customizes existing Web pages by hiding / changing / attaching new elements.
  • Attach Element
    Attach [source] to [target] at [front_or_back].
  • Click
    Click [target].
  • Create Element
    Create [tag] elements using the [value].
  • Hide
    Hide elements in [target].
  • Keyboard
    Type [text] at [inputBox].
  • Set Attribute
    Set [key] attributes of [target] to [new_value].
  • Show
    Show elements in [target].
  • Store
    Store [data] in [permission] storage.

Flow

Manages program structure such as event handling.
  • Trigger
    Trigger the following nodes when [event_source] is loaded, clicked, or changed.
